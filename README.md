# tetris-game
Это тетрис-игра, которую я создала, потому что мне было скучно, еще в 2023 году. В качестве движка использовался Pygame так же, как и в предыдущей игрушке (almost-pacman-game), которую я создала примерно годом ранее.
### Управление:
Стрелки влево и вправо - перемещение фигуры в падении влево или вправо соответственно. Как только фигура коснулась нижней части окна, ее перемещение больше недоступно.
Стрелка вниз - ускорение падения фигуры.
Стрелка вверх - поворот фигуры в падении на 90 градусов.
